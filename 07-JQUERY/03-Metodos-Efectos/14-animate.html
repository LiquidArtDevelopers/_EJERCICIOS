animate()

El método jQuery animado() se utiliza para crear animaciones personalizadas.

Sintaxis:

$("selector").animate({params}.speed.callback);

El parámetro params define las propiedades del CSS para ser animados.

El parámetro de velocidad opcional especifica la duración de del efecto. Puede tomar los siguientes valores: slow, fast o milisegundos.

El pará,etro callback opcional es una función a ejecutar después de la animación completa.
