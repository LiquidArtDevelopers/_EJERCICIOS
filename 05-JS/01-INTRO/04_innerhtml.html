JavaScript posibilidades de visualización

JavaScript puede "visualizar" datos de diferentes maneras:

  • Escribiendo en un cuadro de alerta, utilizando window.alert().
  • Escribir en la salida HTML utilizando document.write().
  • Escribiendo en un elemento HTML, usando innerHTML.

con innerHTML

El método getElementById() accede al primer elemento con el id especificado.
Sintaxis:
document.getElementById("id");

innerHTML nos permite cambiar/insertar el contenido HTML de un elemento.
Sintaxis:
document.getElementById("id").innerHTML ="valor";

Acceder a objeto de estilo de un elemento especificado
Sintaxis:
document.getElementById("nombre").style;
Para establecer las propiedades de estilo de un elemento existente
document.getElementById("nombre").style.color = "red"

Aplicar un estilo de clase
Sintaxis:
document.getElementById("nombre").className;
document.getElementById("myH1").className = "nombreestilo"
