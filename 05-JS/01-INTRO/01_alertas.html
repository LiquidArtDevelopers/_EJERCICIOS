JavaScript posibilidades de visualización

JavaScript puede "visualizar" datos de diferentes maneras:

  • Escribiendo en un cuadro de alerta, utilizando window.alert().
  • Escribir en la salida HTML utilizando document.write().
  • Escribiendo en un elemento HTML, usando innerHTML.

window.alert()

El método de alerta () muestra un cuadro de alerta con un mensaje específico y un botón Aceptar.

Un cuadro de alerta se utiliza si quiere asegurarse de que la información llega hasta el usuario.

Nota: No se debe abusar de este método, ya que evita que el usuario pueda acceder a otras partes de la página hasta que la caja está cerrada.
